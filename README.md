# Erasemine 🧨

Pygame으로 구현한 지뢰찾기(Minesweeper) 게임입니다.

## Features
- Easy / Medium / Hard 난이도 지원
- 난이도별 보드 크기 및 지뢰 개수 적용
- 힌트 기능 (난이도별 제한 횟수)
- 게임 타이머 및 점수 시스템
- 지뢰 개수에 따른 색깔 표시

## Controls
- 좌클릭: 칸 열기
- 우클릭: 깃발 표시
- R 키: 게임 재시작
- 1 / 2 / 3 키: 난이도 변경
- Hint 버튼: 힌트 사용

## How to Run
```bash
pip install pygame
python run.py
